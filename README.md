# QuickShow 🎬

QuickShow ek versatile full-stack application hai jo meri technical transition aur versatility ko darshata hai. Is project ke zariye maine apne coding skills ko diverse domains mein apply kiya hai.

## 🌟 Key Highlights
* **Domain Versatility:** Maine apne skills ko single project se aage badha kar **E-commerce (QuickShow)** jaise domains mein apply kiya hai.
* **MERN Stack:** MongoDB, Express.js, React.js, aur Node.js ka use karke ek scalable architecture taiyaar kiya hai.

## 🚀 Tech Stack
* **Frontend:** React.js (Hooks, Props, Functional Components)
* **Backend:** Node.js & Express.js
* **Database:** MongoDB
* **Version Control:** Git & GitHub

## ✨ Features
* **Product Management:** E-commerce functionality for movie-related shows/products.
* **RESTful APIs:** Secure and efficient backend routes.
* **Responsive Design:** Mobile-friendly user interface.

## 🛠️ Setup Instructions
1. Clone this repository.
2. Run `npm install` in both frontend and backend folders.
3. Use `npm start` to run the development server.
